ゲームの概要
伝統的な花札の要素を簡略化し、「3枚のカードを選んで役を作る」という戦略性にフォーカスした対戦型ゲームです。プレイヤーとCPUが互いのHP（20）を削り合い、先に相手の体力を0にした方が勝利となります。

特徴・こだわった点
舞台月ボーナスシステムで、毎試合ランダムに決まる「舞台月」と同じ月の札を出すと、役の強さにボーナスが加算される仕組みを実装しました。

手札がそろう確率を調整するために、手札5枚の中から最大5枚まで一度だけ引き直しができる機能を搭載し、運要素と戦略性のバランスを調整しました。

直感的にプレイできるようにHTMLやCSSを駆使し、選択中のカードの枠や舞台月のカードの枠に色をつけて、プレイ状況が視覚的に分かりやすいよう設計しました。

使用技術
HTML5:ゲームの構造定義
CSS3:フレックスボックスを用いたレイアウト、アニメーション、役ランク別のカラーリング
JavaScript(Vanilla JS):ゲームロジック（山札管理、役判定アルゴリズム、CPUの思考ロジック、非同期処理によるバトル演出）の実装

実装した役一覧
月下無双 (SSS): 舞台月を3枚揃える最強の役。10点＋貫通ダメージ。
五光 (S): 光札を3枚揃える役。6点。
猪鹿蝶 / 短冊役 (A): 特定の組み合わせで成立。5点。
その他、花見酒・月見酒、三つ揃いなど多数の役を判定ロジックに組み込んでいます。
